OPEN SIM KIT

SIM Kit is a project which aims to make it easier for users to turn their SIM cards into information storage cards. Open SIM Kit (OSK) is a solution for activists, journalists, and anyone else interested in using SIM chips as read/write devices. You can find out more about it here.
